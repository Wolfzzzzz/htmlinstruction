🏛️ 文档结构
<!DOCTYPE html>
HTML5 文档类型声明，告诉浏览器使用标准模式。
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>HTML结构示例</title>
</head>
<body>
    <h1>欢迎学习HTML</h1>
</body>
</html>
<?php
// PHP 动态输出 HTML 结构示例
header('Content-Type: text/html; charset=utf-8');
?>
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>PHP生成HTML结构</title>
</head>
<body>
    <h1><?php echo 'PHP 动态标题: ' . date('Y-m-d'); ?></h1>
    <p>当前时间: <?= date('H:i:s') ?></p>
</body>
</html>

Dialog 对话框

📘 HTML教学弹窗

原生dialog元素 + PHP后端可配合

<dialog id="modal">内容</dialog>
<button onclick="modal.showModal()">打开</button>
<?php
// PHP 动态生成对话框

<button onclick="document.getElementById(\'phpDialog\').showModal()">PHP弹窗</button>';
?>
